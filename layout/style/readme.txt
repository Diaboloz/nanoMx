
ACHTUNG WICHTIG !!
------------------


Bitte ändern sie nicht die Original Systemdateien in diesem Ordner!

Um das Standard Stylesheet anzupassen, erstellen sie in diesem 
Ordner eine zusätzliche css Datei die ihre Änderungen enthält.

Der Dateiname muss mit "custom" beginnen und die Dateiendung 
muss ".css" sein. 
z.B.: custom.irgendetwas.css


NOTE IMPORTANTE !!
------------------


Veuillez ne pas modifier les fichiers originaux du système présents dans ce dossier!

Pour personnaliser la feuille de style par défaut, il suffit simplement de créer/placer
dans le présent dossier un fichier css supplémentaire contenant vos modifications personnelles.
Ce nouveau fichier "surpassera" alors l'ancien fichier système par défaut.

Le nom du fichier doit commencer par "custom" et l'extension du fichier doit être ".css"

Par exemple, si vous souhaitez remplacer le fichier "default.css" alors votre nouveau fichier 
doit s'appeler "custom.default.css"
