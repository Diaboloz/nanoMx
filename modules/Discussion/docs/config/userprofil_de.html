2.2 Profil - Einstellungen für normale User

Hier werden kurz die verschiedenen Einstellungsmöglichkeiten der User dargestellt




Theme
Hier haben Sie die Möglichkeit zwischen verschiedenen Themes des Forums zu wählen. "default" bedeutet, das zur Darstellung des Forums die Standard-Farben des pragmaMx-Systems verwendet werden. Zwischen wie vielen Themes Sie wählen können ist abhängig davon, was die Administratoren für Sie freigegeben haben.


Sprache
Hier können Sie Ihre gewünschte Sprache für die Forumsdarstellung einstellen.


Topics pro Seite
Mit diesem Zahlenwert legen Sie fest, wie viele Topics / Beiträge bei Ihnen pro Seite in der Forumsansicht dargestellt werden.


Posts pro Seite
Mit diesem Zahlenwert legen Sie fest, wie viele Posts / Antworten bei Ihnen pro Seite angezeigt werden, wenn Sie ein Topic / Beitrag lesen.


Zeitformat
Legt fest, ob für Sie die Zeitangaben im 24h-Format oder im 12h-Format dargestellt werden.

Außerdem können Sie hier einstellen, wie viele Stunden "neue Beiträge seit Ihrem letzten Besuch" unter der Menüoption "neue Posts" im Header des Forums angezeigt werden. Ist die Zeitspanne abgelaufen wird die Ansicht zurückgesetzt.


Datumsformat
Legt die Datums-Darstellung für Ihr Profil fest


Optionen
Hiermit legen Sie diverse Einstellungen fest:

Email-Adresse anderen zeigen?
Wenn Sie diesen Hacken setzen erscheint u.A. unter Beiträgen von Ihnen ein Button "E-Mail", mit dem andere Benutzer die Möglichkeit haben, Ihnen Email-Nachrichten zu schicken. Die Email-Adresse, die hierfür verwendet wird, ist die Email-Adresse, die Sie zur Registration am pragmaMx-System angegeben haben.

Sollen erweiterte Informationen von Ihnen gezeigt werden wenn verfügbar?
Ist diese Option aktiviert werden in Ihrem Forums-Profil alle Informationen von Ihnen angezeigt, die sie im pragmaMx-System unter "Ihre Informationen" angegeben haben.

Newsletter vom Admin empfangen?
Ist diese Option aktiviert erhalten Sie Forums-Newsletter von Ihrem Systemadministrator. Bitte beachten Sie, das diese Option nichts mit dem Newsletter des pragmaMx-Systems zu tun hat, d.h. wenn Sie für den Newsletter des pragmaMx-Systems eingetragen sind erhalten Sie trotzdem keine Forums-Newsletter und umgekehrt.

Sollen andere User Ihnen Private Messages senden können?
Wenn Sie diese Option aktivieren erscheint unter Beiträgen von Ihnen ein "u2u"-Button (User to User). Dieser ermöglicht es anderen Mitgliedern Ihnen Private Messages direkt aus dem Forum heraus zu senden. Es wird hierfür der Private Messages-Dienst des pragmaMx-Systems genutzt.

BITTE BEACHTEN: Diese Option ist nur verfügbar, wenn Ihr Administrator dies auf dem System aktiviert hat. Ist diese Option nicht vorhanden, ist dieser Service deaktiviert.

Zeitoffset
Hier haben Sie die Möglichkeit Ihre lokale Zeitzone einzustellen. Referenzzeit ist die GMT-Zeit, die Einstellung erfolgt in Differenz-Stunden zur GMT-Zeit

BITTE BEACHTEN: Diese Option ist nur verfügbar, wenn Ihr Administrator dies auf dem System aktiviert hat. Ist diese Option nicht vorhanden, ist dieser Service deaktiviert.


Signatur
Hier haben Sie die Möglichkeit, eine Signatur für Nachrichten zu erstellen. Links neben dem Textfeld wird außerdem noch angezeigt, ob BB-Code und HTML-Code in der Signatur verwendet werden können oder ob dies auf dem System für Signaturen deaktiviert ist.
