2.2 Profil - Settings for Users

The different setting possibilities of the user account are represented here briefly




Theme
Here you have the possibility to choose a different forum-theme for your user profile


Language
Here you can change you default language for the forum


Topics Per Page
With this number you set how many topics are shown for you on one page


Posts Per Page
With this number you set how many posts are shown for you on one page


Time Format
Here you set a 24h-timeset or 12h-timeset for you profile

With the next setting you adjust, how long "new posts since you last visit" are displayed unter the "new posts"-Link in the header of the forum.


Date Format
Here you set the date format for your profile


Options
Here you set multiple things:

Make e-mail address viewable by others?
If active, other users can see the email-address you used to reg onto the nuke system.

Should advanced userinformation been shown if available (icq, location ....)?
If active, all informations available under the "your account"-settings are shown about you.

Receive newsletters from the admin?
If active, you will receive the forums newsletter. This newletter is independent from the pragmaMx system.

Should other users are allowed to send you Private Messages out of the forum?
If active, a "u2u"-button will appear under threads and posts from you. With this button other users can send you Private Messages over the pragmaMx system.

PLEASE NOTE: this option is only available if your administrator activates this for you.

Time Offset
Here you can set you timezone. Setting is taken in difference hours between your local time and the GMT-timezone.

PLEASE NOTE: this option is only available if your administrator activates this for you.

Signature
Here you can modify you signature. At the left side you can see if BB-Code and HTML-Code is allowed to use in the signature.
