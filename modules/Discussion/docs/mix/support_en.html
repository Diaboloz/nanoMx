Support

This version of the mxBoard module is free of charge. Support for it is granted only online within the support forum. There exists no claim to support, this service is based on the voluntary performance of the authors and other members of pragmaMx.org.

It is pointed out to hereby particularly that no questions which are covered by this manual are answered. Look up here so please first. In addition, we ask for it in the support forum :
  1. using the search function to find contributions with the same problem
  2. download updates and install them if available
If this doesn't help you, please create a new thread including the following informations:
  1. Forums-Version (you can find at the end of the "settings"-page from the administrator menu)
  2. version of your pragmaMx system
  3. PHP-Version
  4. the provider where you have the problems
  5. link to the page where we can see the problem
  6. a detailled error-report (especially how the error can be produced ...)
Please note, that this is the only way for us to deliver good support for your problems without spending too much of our "free of charge"-time with it. If you can't deliver us any of this informations, please TELL THIS TO US in the thread, otherwise it might be that we will not answer your question, sorry.

Use of the mxBoard-module is done on your own risk. You can't make the authors liable for any problems you might have during the installation / the upgrade or the normal operation.

The module with all Scripten was tested in detail and found to be good after the best conscience. However, it never can be excluded that there might be problems depending on PHP-version conflicts or overlooked bugs wich can cause data-loose.
