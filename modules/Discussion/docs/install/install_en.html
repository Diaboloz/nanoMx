1.2 Installation

The following three installation possibilities are conceivable:
  1. 1st installation - there is no mxBoard-forum onto your system
  2. extension installation - another mxBoard forum shall be installed in addition to another mxBoard (s) (this concerns only mxBoard forums, no other forums!)
  3. Upgrade - upgrade the "intern XMB-Forum for PHPNuke 5.5 from fsz- & maaX-dESIGN"to the mxBoard-forum

Note: The option instruction always refers to the module name "mxBoard". If you should use another module name, then you must change all "mxBoard" calls against your module name. We consider, that you have read the notes "before the installation"!

1. 1st installation
  1. You must be logged in as administrator on your system
  2. Start the installation-script with this link: http://www.YourDomain.com/modules.php?name=mxBoard&file=install (if you have any problems with this script, please read the part at the end of this page!).
  3. Coose your desired language and the operation "install"..
  4. "Step 1". Now you see the settings of you pragmaMx system. Please don't change anything except you know what you do! Now click "Next".
  5. "Step 2". If the installation-script get access to your database, you can click "next" and the forum will be insert into your database.
  6. "Step 3". Database was created. Now there will be some default-values (Smilies, Themes ...) be insert into your database. Click "Next"
  7. If everything worked well, you should be forwarded to the new forum. Installation is completed hereby.
  8. Please read now the part "After the installation".


2. Extension installation

"mxBoard2" is accepted as an additional module name here. This name has to be replaced by your module name if necessary.

  1. You must be logged in as administrator on your system
  2. Start the installation-script with this link: http://www.YourDomain.com/modules.php?name=mxBoard&file=install (if you have any problems with this script, please read the part at the end of this page!).
  3. Coose your desired language and the operation "install"..
  4. "Step 1". Now you see the settings of you pragmaMx system. Please don't change anything except you know what you do! Only thing you HAVE TO CHANGE is the "mxBoard Prefix". If you allready have an mxBoard-Modul running on your system, the prefix "mxBoard" will be used by the first forum. So change this prefix into something unused, lets say "mxBoard2". Now click "Next".
  5. "Step 2". If the installation-script get access to your database, you can click "next" and the forum will be insert into your database.
  6. "Step 3". Database was created. Now there will be some default-values (Smilies, Themes ...) be insert into your database. Click "Next"
  7. If everything worked well, you should be forwarded to the new forum. Installation is completed hereby.
  8. Please read now the part "After the installation".


3. Upgrade

  1. You must be logged in as administrator on your system.
  2. Deactivate the XForum-module on your system (you do this in the administratormenu of your system). This keeps "normal users" away from your forum while upgrading it to the mxBoard, but it doesn't keep other administrators of your system away from it. So make sure noone access your forums-db until upgrade to mxBoard is complete!
  3. Start the installation-script with this link: http://www.YourDomain.com/modules.php?name=mxBoard&file=install (if you have any problems with this script, please read the part at the end of this page!).
  4. Coose your desired language and the operation "install"..
  5. "Step 1". Now you see the settings of you pragmaMx system. Please don't change anything except you know what you do! DO NOT CHANGE THE "mxBoard Prefix"! Now click "Next".
  6. "Step 2". If the installation-script get access to your database, it points you to backup your database. PLEASE DO SO before you click "Next". Now the old database will be changed to the new database-structure of the mxBoard-Module.
  7. "Step 3". Database was created. Now there will be some default-values (Smilies, Themes ...) be insert into your database. Click "Next"
  8. If everything worked well, you should be forwarded to the new forum. Installation is completed hereby.
  9. Please read now the part "After the installation".
