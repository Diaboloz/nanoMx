1.2 Installation

Folgende drei Installationsmöglichkeiten sind denkbar:
  1. Neuinstallation - es gibt noch kein mxBoard-Forum auf Ihrem System
  2. Erweiterungsinstallation - ein weiteres mxBoard-Forum soll zusätzlich zu einem anderen mxBoard(s) installiert werden (dies betrifft nur mxBoard-Foren, keine anderen Foren!)
  3. Upgrade - das "interne XMB-Forum für PHPNuke 5.5 von fsz- & maaX-dESIGN" auf das mxBoard-Forum upgraden

HINWEIS: Die Installationsanleitung bezieht sich immer auf den Modul-Namen "mxBoard". Sollten Sie einen anderen Modul-Namen verwenden, dann müssen Sie alle "mxBoard"-Aufrufe gegen Ihren Modulnamen ändern. Es wird davon ausgegangen, das Sie die Hinweise "Vor der Installation" gelesen haben!


1. Neuinstallation
  1. Sie müssen als Administrator auf Ihrem System eingelogged sein.
  2. Starten Sie das Installations-Script mit folgendem Aufruf: http://www.IhreDomain.de/modules.php?name=mxBoard&file=install (sollten Sie mit dem Aufruf des Installations-Scriptes Probleme haben, dann lesen Sie bitte den Abschnitt am Ende dieser Seite!).
  3. Wählen Sie Ihre gewünschte Setup-Sprache und die Operation "Install" aus.
  4. "Schritt 1". Jetzt sehen Sie die Einstellungen Ihres pragmaMx-Systems. Ändern Sie hierdrann bitte nichts, wenn Sie nicht genau wissen, was sie tun! Unten können sie zwischen drei Standard-Sprachen und diversen Themes wählen, die das Forum als "default"-Werte nutzt. Diese Einstellungen können später noch im Administrationsmenü des Forums geändert werden. Klicken Sie dann auf "Weiter"
  5. "Schritt 2". Wenn Verbindung zu Ihrer Datenbank hergestellt werden konnte können Sie auf "Weiter" klicken, das Forum wird dann in die SQL-Datenbank eingefügt.
  6. "Schritt 3". Die Forums-DB wurde angelegt, es werden jetzt default-Werte in die DB eingetragen (Smilies, Themes ....). Klicken Sie auf "Weiter"
  7. Wenn alles funktioniert hat sollten Sie sich jetzt auf der Startseite Ihres neuen Forums-Moduls befinden. Die Installation ist hiermit abgeschlossen.
  8. Bitte jetzt die Punkte unter "Nach der Installation" beachten.


2. Erweiterungsinstallation

Hier wird als zusätzlicher Modulname "mxBoard2" angenommen. Dieser Name ist ggf. durch Ihren Modulnamen zu ersetzen.
  1. Sie müssen als Administrator auf Ihrem System eingelogged sein.
  2. Starten Sie das Installations-Script mit folgendem Aufruf: http://www.IhreDomain.de/modules.php?name=mxBoard2&file=install (sollten Sie mit dem Aufruf des Installations-Scriptes Probleme haben, dann lesen Sie bitte den Abschnitt am Ende dieser Seite!).
  3. Wählen Sie Ihre gewünschte Setup-Sprache und die Operation "Install" aus.
  4. "Schritt 1". Jetzt sehen Sie die Einstellungen Ihres pragmaMx-Systems. Ändern Sie hierdrann bitte nichts, wenn Sie nicht genau wissen, was sie tun! Das Einzige, was sie ändern müssen, ist das "mxBoard Prefix". Dieses ist standardmäßig auf "mxBoard" gesetzt. Wenn sie jetzt aber schon ein mxBoard installiert haben wird dieses Prefix wahrscheinlich schon vergeben sein (es sei denn, Sie haben vom XForum auf das mxBoard geupdated. Dann ist der XForum-Tabellenname "xmb" bestehen geblieben und "mxBoard" wurde als Prefix noch nicht belegt). Ändern Sie dieses "mxBoard Prefix" am besten auf Ihren Modulnamen, hier also "mxBoard2" (bitte beachten das diverse Prefixe schon vom pragmaMx-System belegt sind. Daher ist ein Prefix-Name wie "mxBoard2" am sichersten um Verwechselungen zu vermeiden). Unten können sie zwischen drei Standard-Sprachen und diversen Themes wählen, die das Forum als "default"-Werte nutzt. Diese Einstellungen können später noch im Administrationsmenü des Forums geändert werden. Klicken Sie dann auf "Weiter"
  5. "Schritt 2". Wenn Verbindung zu Ihrer Datenbank hergestellt werden konnte können Sie auf "Weiter" klicken, das Forum wird dann in die SQL-Datenbank eingefügt.
  6. "Schritt 3". Die Forums-DB wurde angelegt, es werden jetzt default-Werte in die DB eingetragen (Smilies, Themes ....). Klicken Sie auf "Weiter"
  7. Wenn alles funktioniert hat sollten Sie sich jetzt auf der Startseite Ihres neuen Forums-Moduls befinden. Die Installation ist hiermit abgeschlossen.
  8. Bitte jetzt die Punkte unter "Nach der Installation" beachten.

3. Upgrade
  1. Sie müssen als Administrator auf Ihrem System eingelogged sein.
  2. Deaktivieren Sie im Administrationsmenü des pragmaMx-Systems unter "Module" das "XForum"-Modul. Dies verhindert das während der Konvertierung und Installation Zugriffe auf die DB des XForums erfolgen. Aber bitte beachten Sie, das dies nur "normale User" davon abhält auf das XForum zuzugreifen, Administratoren haben nach wie vor Zugriff auf dieses Modul. Stellen Sie also bitte sicher das kein Administrator auf das XForum wärend der Installation zugreift! Es könnte sonst zu Datenverlusten kommen.
  3. Starten Sie das Installations-Script mit folgendem Aufruf: http://www.IhreDomain.de/modules.php?name=mxBoard&file=install (sollten Sie mit dem Aufruf des Installations-Scriptes Probleme haben, dann lesen Sie bitte den Abschnitt am Ende dieser Seite!).
  4. Wählen Sie Ihre gewünschte Setup-Sprache und die Operation "Upgrade vom XForum -> mxBoard" aus.
  5. "Schritt 1". Jetzt sehen Sie die Einstellungen Ihres pragmaMx-Systems. Ändern Sie hierdrann bitte nichts, wenn Sie nicht genau wissen, was sie tun! UNTER KEINEN UMSTÄNDEN DAS mxBoard Prefix ÄNDERN! Dies ist das Prefix Ihres alten XForums und MUSS bestehen bleiben für ein Update. Unten können sie zwischen drei Standard-Sprachen und diversen Themes wählen, die das Forum als "default"-Werte nutzt. Diese Einstellungen können später noch im Administrationsmenü des Forums geändert werden. Klicken Sie dann auf "Weiter"
  6. "Schritt 2". Wenn Verbindung zu Ihrer Datenbank hergestellt werden konnte ist soweit alles dazu bereit, Ihre XForum-DB in die mxBoard-DB zu konvertieren. Das Installations-Script fordert Sie jetzt auf ein Backup der pragmaMx-DB zu machen und bietet Ihnen dazu einen Backup-Button an. Es wird dringenst empfohlen dieses Backup anzulegen falls Sie vorher Ihre Datenbank nicht gesichert haben. Bitte warten Sie auf jeden Fall, bis der Download komplett ist BEVOR sie auf "Weiter" klicken.
  7. "Schritt 3". Die Forums-DB wurde konvertiert, es werden jetzt default-Werte in die DB eingetragen (Smilies, Themes ....). Klicken Sie auf "Weiter"
  8. Wenn alles funktioniert hat sollten Sie sich jetzt auf der Startseite Ihres neuen Forums-Moduls befinden. Die Installation ist hiermit abgeschlossen.
  9. Bitte jetzt die Punkte unter "Nach der Installation" beachten.
