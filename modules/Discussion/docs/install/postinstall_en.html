1.3 After the installation

Please still take the following points which apply to your installation into account after a successful installation:

Activation of the new forum module:

After the installation the new forum is obtainable only for administrators of the pragmaMx-Systems. To make it for all users accessibly, module "mxBoard" must be activated under "modules" in the administration menu of the pragmaMx system (or the module name which you have chosen). You also adjust here, which user groups may access this module.


After an upgrade of the XForum to the mxBoard:

Delete the old "XForum"-module from your "modules"-directory. The old forum is no longer compatieble with the database-structure, so there is no need to keep it.

Activating the mxBoard center block:

If you would like to use the forums-center-block onto you initial page, activate the "mxBoard_Center"-block in the administratormenu of your pragmaMx system.

If you changed the module name from "mxBoard" to something else, you also have to do this into the "mxBoard_Center"-block.

You can also use more than one center-block on the initial page. All you have to is making a copy of the "block-mxBoard_Center.php" in the "blocks"-directory of your system and changing the module-name into it.
