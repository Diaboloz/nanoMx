1.3 Nach der Installation

Nach erfolgreicher Installation bitte noch die folgenden Punkte beachten, die für Ihre Installation zutreffen:

Aktivierung des neuen Forum-Modules:

Nach der Installation ist das neue Forum nur für Administratoren des pragmaMx-Systems erreichbar. Um es für alle Benutzer zugänglich zu machen muss im Administrationsmenü des pragmaMx-Systems unter "Module" das Module "mxBoard" aktiviert werden (oder ggf. der Modulname, den Sie gewählt haben). Hier stellen Sie auch ein, welche Usergruppen auf dieses Modul zugreifen dürfen.


Nach einem Upgrade vom XForum auf das mxBoard:

Löschen Sie den alten XForum-Modulordner aus dem Ordner "modules" nachdem Sie sich eine Kopie auf Ihren lokalen Rechner gesichert haben. Das alte Forum ist mit der neuen Datenbankstruktur nicht mehr kompatiebel und daher funktionsunfähig. Es besteht daher keinerlei Grund, dieses Modul weiterhin zu behalten. Ganz im Gegenteil, es könnte Probleme verursachen. Bitte löschen Sie dieses Modul natürlich nicht, wenn Sie das mxBoard parallel zum XForum installiert haben (also die Option "Install" und nicht "Upgrade" gewählt haben!)

Aktivieren des mxBoard-Center-Blocks:

So Sie für Ihr mxBoard den Center-Block auf der Startseite des pragmaMx-Systems nutzen möchten gehen Sie in das Administrator-Menü des pragmaMx-Systems und aktivieren Sie unter "Blöcke" den Block "mxBoard_Center". Dieser Block kann weder links noch rechts angezeigt werden, sondern nur im Center.

Sollten Sie schon den Center-Block für das XForum benutzt haben, so ist dieser nach wie vor kompatiebel mit dem mxBoard, jedoch hat der neue Center-Block mehr Funktionen als der alte Block. Es liegt an Ihnen, welchen Block Sie nutzen möchten. Dies funktioniert aber nur nach einem Upgrade des XForum auf das mxBoard, ansonsten läuft der alte Center-Block nicht mit dem neuen Forum ohne Modifikationen!

Sollten Sie einen anderen Modul-Namen als mxBoard benutzen müssen Sie diesen im Center-Block ändern.

Sie können auch mehrere Center-Blöcke für mehrere Foren benutzen. Sie müssen dann nur die Datei "blocks/block-mxBoard_Center" kopieren und umbenennen, z.B. in "blocks/block-mxBoard2_Center" und den Modul-Namen in diesem zweiten Block entsprechend anpassen. Dann aktivieren Sie den Block "mxBoard2_Center" wie oben beschrieben.
