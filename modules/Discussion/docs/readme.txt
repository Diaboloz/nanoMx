mxBoard-Forum Version: 1.2
--------------------------

Die Dokumentation des mxBoard und weitere Hinweise zur Installation 
finden sich hier: http://www.pragmamx.org/doku.php?id=mxboard


Installationsvoraussetzungen:
-----------------------------
* pragmaMx ab 2.2.x
* PHP ab Version 5.4.x
* 


Neuinstallation:
---------------
* Es wird das Downloadpaket für Neuinstallation benötigt
* Alle Dateien aus dem html Verzeichnis auf den Server kopieren. Die 
  Verzeichnisstruktur muss dabei erhalten bleiben.
* Im pragmaMx-System als System-Administrator einloggen.
* Im Adminbereich >> Cache zurücksetzen
* Über die Moduladministration das neue Modul "mxBoard" aktivieren und 
  die weiteren Einstellungen treffen.
* Die Installationsroutine des Moduls wie folgt aufrufen:
  - http://www.DEINEDOMAIN.de/modules.php?name=mxBoard&file=install
  und den Anweisungen folgen.
* Die Installationsdateien brauchen nicht gelöscht zu werden.
* Nach der Installationsroutine wird man zu den Einstellungen geleitet.
  Diese gut überprüfen und auf jeden Fall abspeichern
* Fertig...
